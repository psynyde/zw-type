## zw-type
zig port of the [wl-ime-type](https://gitlab.freedesktop.org/emersion/wl-ime-type) which uses [input-method-unstable-v2](https://gitlab.freedesktop.org/wlroots/wlroots/-/blob/9119b8aa851c413b79295f2a913124f0eed536d4/protocol/input-method-unstable-v2.xml) to type text in wayland.

### Usage:
```
$ zw-type [options] <text>

Options:
  -h, --help           Show this help message
  -s, --seat <name>    Select seat by name
      --list-seats     List available seats and exit
```
